# Facility Location

Clean workspace for the facility location project.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. (Optional) Register Jupyter kernel:

```bash
python -m ipykernel install --user --name facility-location --display-name "Python (facility-location)"
```

## Project structure

- `data/raw`: original source files
- `data/processed`: cleaned and transformed datasets
- `notebooks`: exploratory and analysis notebooks
- `outputs/figures`: charts and maps
- `outputs/tables`: exported tables
